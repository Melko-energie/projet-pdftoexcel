1 Structure du dossier ZIP

Votre archive doit contenir deux répertoires racines :

  • mails/ : Contenant les documents principaux.
  • proof/ : Contenant les pièces justificatives.

2 Convention de nommage

Chaque fichier doit commencer par son numéro d'index (ex: 001_tax_report.pdf). Ce préfixe assure l'alignement entre le courrier et sa preuve.
